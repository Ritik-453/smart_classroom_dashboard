# 🏫 Smart Classroom Automation Dashboard

A modern Smart Classroom Automation Dashboard built using HTML, CSS, JavaScript, and Chart.js for real-time classroom monitoring and automation.

---

## 🚀 Features

✅ Secure Login System  
✅ Real-Time Sensor Monitoring  
✅ PIR Motion Detection  
✅ Temperature & Humidity Tracking  
✅ Automatic AC Control  
✅ Live Alerts System  
✅ Dynamic Data Table  
✅ Real-Time Graph Analytics  
✅ Responsive Dashboard UI  

---

## 📷 Dashboard Modules

- Dashboard Overview
- PIR Sensor Monitoring
- DHT11 Sensor Analytics
- AC Automation
- Live Classroom Alerts
- Sensor Data Table
- Real-Time Graph Visualization

---

## 🛠 Technologies Used

- HTML5
- CSS3
- JavaScript
- Chart.js
